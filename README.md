# 🌌 Eesh Chitrarth Sharma — Portfolio

A galaxy-themed, fully responsive portfolio website. Pure HTML/CSS/JS — no build step, no dependencies.

## Features

- **Starfield canvas** with three parallax depth layers that react to mouse movement
- **Constellation lines** that connect stars near the cursor
- **Shooting stars**, drifting nebula blobs, custom glowing cursor with trailing ring
- **3D tilt cards** with spotlight tracking, magnetic buttons and social icons
- Typewriter hero, animated stat counters, scroll-reveal sections
- Real code snippets pulled from my public repos ([SanBrewCafe](https://github.com/chitrarth11/SanBrewCafe), [ChitrarthAthletics](https://github.com/chitrarth11/ChitrarthAthletics), [Chitrarth-Dance-Academy](https://github.com/chitrarth11/Chitrarth-Dance-Academy))
- Responsive across desktop / tablet / mobile, honors `prefers-reduced-motion`, touch-friendly fallbacks
